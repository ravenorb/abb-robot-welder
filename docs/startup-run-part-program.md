# Startup and Run Procedure

## Startup checklist
1. Power on the controller and confirm all safety interlocks are active.
2. Verify the correct tool (`tool0`) and wire are installed.
3. Confirm the BullsEye target is clean and reachable.
4. Load `WeldProgram.mod` and start `WeldProgram.main`.

## Run a part program
1. Select the part number (1â€“6).
2. Confirm the weld slots are populated (optional review step).
3. Use **Review Weld** to dry-run any new or modified welds.
4. Switch to the appropriate welding mode and begin production.

## When to re-run setup
- After any fixture change: run **Setup Part**.
- After any torch adjustment: re-run **Teach Orientation**.
