# Program Selection Map

```
MainModule.main
 ├─ Production
 │   ├─ Run
 │   ├─ Preview
 │   └─ Utilities Menu
 ├─ Setup
 │   ├─ Work Origin
 │   ├─ Preset Positions
 │   ├─ Weld Config
 │   └─ Speeds
 ├─ Teaching
 │   ├─ Edit Origin
 │   └─ Wipe Origin
 ├─ Position
 └─ Load/Save
```

## Notes
- Use **Work Origin** to record origin base/limit/Z-plane points.
- **Edit Origin** drives add/view/delete weld workflows.
- **Weld Config** stores torch angle and preset data.
