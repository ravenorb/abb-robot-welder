# Program Selection Map

```
WeldProgram.main
 ├─ Select Part (1–6)
 └─ Select Action
     ├─ Teach Orientation
     ├─ Setup Part
     ├─ Add Weld
     ├─ Review Weld
     ├─ Edit Weld
     └─ Delete Weld
```

## Notes
- Teach Orientation stores the pose for a specific orientation.
- Setup Part records part reference points and updates the work object.
- Add Weld stores the next free slot automatically.
